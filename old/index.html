En esta web aprenderemos a crear juegos simples de principio a fin usando Cocos2d-x, un framework multiplataforma que nos permite hacer nuestro juego en C++ y compilarlo para Android, IOS, Mac, Linux, Win32 y Windows store (universal apps) entre otras plataformas. Se asume que el lector tiene experiencia con algún lenguaje de programación como Java.

En la página web de Cocos podemos encontrar algunos recursos para su aprendizaje como son estos dos recursos que dejo aquí. Recomiendo la Guía del programador pues muestra cómo se usa cada componentes de forma individual. Los recursos están en inglés, sin embargo aquí trataremos aunque no todos los elementos, si una gran variedad, en español, con lo que la revisión de estos enlaces puede hacerse después de los tutoriales de esta web.

  • API V3.x
  • Guía del programador

Cualquier juego realizado aquí, se ha realizado sin ánimo de lucro con fines meramente educativos, los derechos de los recursos artísticos empleados (música imágenes) pertenecen a sus creadores. Para comenzar a aprender acerca de cocos, dirígete a la pestaña Tutoriales de esta web.

Los juegos los puedes encontrar en la Windows store para Windows phone y en la play store de Android.
