Capítulo 10 - Introducción

En este capítulo veremos como utilizar y crear sistemas de particulas como los empleados cuadno una bala golpea al jugador, hay infinidad de herramientas pero una vez mas de pago, sin embargo para sistemas de particulas en 2d con cocos podemos utilizar la web particle2dx, una web para generar sistemas de particulas compatibles con cocos. Además de aprender a usar sistemas de particulas crearemos algunos upgrades y nuevos enemigos. Las imagenes para estos nuevos componentes ya están integradas en los spritesheets que se descargarón en el capítulo 8.
